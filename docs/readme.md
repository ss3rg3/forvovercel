# Setup

- Install packages

  ```
  pnpm install
  ```

- Add path to local bin directory.

  ```bash
  export PATH=$PATH:/home/cc/Desktop/Programming/repos/forvovercel/node_modules/.bin
  ```

- Login

  ```
  vercel login
  ```

- Deploy (link project and such)

  ```bash
  vercel			# for preview, separate URL will be provided
  vercel --prod	# deploy
  ```



# Development

- Run locally

  ```
  vercel dev
  ```
